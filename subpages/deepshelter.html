Project Description

Rebranding for a local project which aims at creating "a collaboration between health care professionals and artists in hospitals with the scope of creating or enabling artwork that benefits the patient." All works in this project were done in collaboration with three colleagues; Kimberley Agius, Maria Elena Galea and Maria Fenech.
